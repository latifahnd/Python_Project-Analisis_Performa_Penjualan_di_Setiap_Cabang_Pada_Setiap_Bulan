Visualisasi berupa grafik untuk masing-masing parameter perhitungan melalui slicing dan resampling untuk mendapatkan perhitungan rata-rata bulanan untuk masing-masing parameter perhitungan.¶

In [38]:
#Membuat figure canvas dan axes untuk lima line plots.
figure,axes=plt.subplots(5, 1, figsize =(9,20))

#Slicing index
index=pd.IndexSlice['order', 'customer', 'product', 'brand', 'GMV']
for i, measurement in enumerate(provinsi_group.columns):
    #Slicing data untuk masing-masing measurement
    by_measurement=unstack_provinsi_group.loc[:, index[i]]
    #Melakukan resampling pada data untuk perhitungan rata-rata bulanan
    monthly_by_measurement_mean=by_measurement.resample('M').mean()
    #Plotting monthly_by_measurement_mean
    monthly_by_measurement_mean.plot(title='Average Daily ' + measurement + ' Size in Month View for 5 Provinces', ax=axes[i])
    axes[i].set_ylabel('Average ' + measurement + ' size')
    axes[i].set_xlabel('month')
    
plt.tight_layout()
plt.show()
